# LibzipSwift

### 关于

LibzipSwift 是基于 Libzip 封装用于macOS 和 iOS上对 zip 进行解压缩的 Framework套件. 
它透过 Libzip 提供一下功能：
- 列举 zip 中档案清单，包括概要的属性
- 删除zip 中的档案，包括复写现有档案
- 解压缩 Zip， 支持保留Unix 档案权限 
- 支持对 Winzip 的中文乱码校对

### 安装

使用 Swift Package Manager
